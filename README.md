## 🚀 Featured Projects

### [CartierHQ — AI Command Center](https://github.com/cartiercreative8-art/cartier-hq)

### [CartierOS — Client Automation Suite](https://github.com/cartiercreative8-art/cartier-os)
Productized 4-workflow n8n system deployed to paying clients. Handles lead capture, CRM logging, Slack alerts, daily morning briefings, and automated follow-up. Deploys in under 2 hours from a single template.

11-agent architecture powering CartierCreative's internal operations. Each agent owns a distinct function: Atlas (lead scraping), Hermes (cold outreach), Prometheus (qualification), Solus (briefing), Midas (revenue), and more.

End-to-end lead intelligence system. Apify scrapes Google Maps → Claude AI qualifies leads → auto-imports to CRM → Slack notification. Result: 117 leads sourced, 31 calls made, 3 demos booked. 9.7% call-to-demo rate with zero ad spend.

### [CartierOS Proposal System — AI Sales Automation](https://github.com/cartiercreative8-art/cartier-proposal-system)
AI-generated, prospect-personalized sales proposals. Pulls live business data per target (Google rating, domain status, revenue-at-risk), writes a custom narrative, and outputs a professional PDF. Closes deals.
